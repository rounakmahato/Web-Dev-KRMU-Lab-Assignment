# Capstone Project: Travel Agency Website

**Student Name:** Subhranil Dutta  
**Theme:** Travel Agency Website  

## Project Overview
This project is a fully responsive, semantic HTML5 and CSS3 website for a fictional travel agency named "Subhranil's Travels". It demonstrates modern web design principles, including Flexbox, CSS Grid, and responsive media queries.

## Features
- **Semantic HTML5 Structure**: Uses `<header>`, `<nav>`, `<section>`, `<article>`, `<footer>` for better accessibility and SEO.
- **Responsive Design**: Adapts to Desktop, Tablet, and Mobile screens using media queries.
- **CSS Grid & Flexbox**: Used for complex layouts like the tour packages grid and image gallery.
- **Interactive Elements**: Hover effects, smooth scrolling, and a contact form.
- **High-Quality Imagery**: Includes 20+ high-resolution landscape images.

## Sections Implemented
1.  **Header**: Logo and Navigation menu.
2.  **Hero Section**: Full-screen banner with a call-to-action.
3.  **Popular Destinations**: Grid layout showcasing top locations.
4.  **Featured Tours**: Detailed cards with pricing and ratings.
5.  **Gallery**: A visual grid of travel experiences.
6.  **Testimonials**: Customer reviews.
7.  **Contact Form**: Inquiry form with validation.
8.  **Footer**: Quick links and social media icons.

## How to Run
1.  Clone or download the repository.
2.  Navigate to the `CAPSTONE PROJECT WEB DEV FINAL` folder.
3.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
